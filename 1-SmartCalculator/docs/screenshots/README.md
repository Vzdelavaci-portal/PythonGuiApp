# Screenshots

Do této složky vlož screenshoty aplikace. Doporučené názvy:

```text
calculator-dark.png
function-graph.png
converters.png
calculator-neon.png
```

Pro konzistentní prezentaci používej stejnou velikost okna, ideálně poměr
stran 16:10 nebo 16:9. Před pořízením snímku odstraň z historie případné
osobní nebo testovací údaje.

Po vložení hlavního screenshotu odkomentuj připravený Markdown obrázek
v souborech `README.md` a `README.en.md`.
